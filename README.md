# Creator-Catalyst 🚀
### Growth Intelligence for Content Creators

クリエイターの成長を加速させるための分析プラットフォームです。YouTubeやnoteのCSVデータをアップロードするだけで、統計的に有意な「スパイク（急上昇）」を自動検出し、次に行うべき具体的な3つのアクションを提案します。

## ✨ 特徴
- **汎用データインポーター**: カラム名が「Views」でも「ビュー数」でも自動的に判別し、正規化します。
- **統計的スパイク検知**: 移動平均からの乖離を算出し、たまたまの変動ではない「真の成長の特異点」を特定します。
- **具体的アクション提案**: 「CTRを上げましょう」といった抽象論ではなく、データの傾向に基づいた具体的な戦術を出力します。
- **プロフェッショナルUI**: Streamlitによる、ダークモードベースの洗練されたダッシュボード。

## 🛠️ セットアップ

1. 必要なライブラリのインストール:
   ```bash
   pip install -r requirements.txt
   ```

2. アプリケーションの起動:
   ```bash
   streamlit run app.py
   ```

## 📊 CSVフォーマット
以下のカラムが含まれているcsvに対応しています（一部欠けていても推論します）。
- **日付**: Date, 日付, day...
- **数値**: Views, ビュー数, 視聴回数, pv...
- **コンテンツ名**: Title, タイトル, 動画...
